I build systems that turn intelligence into action: autonomous agents, research pipelines, remote coding infrastructure, and small products that make messy workflows usable.

### Current focus

- **Agent operating systems** — memory, tools, queues, verification, and feedback loops for autonomous work.
- **Local-first AI products** — software that keeps context close, useful, and owned by the user.
- **Remote coding infrastructure** — workstation workflows that survive laptops, phones, bad Wi‑Fi, and long-running agent jobs.
- **Productized research** — turning market/technical sensing into shippable bets instead of dusty notes.

### Operating thesis

- agents need memory, verification, and taste — not just prompts
- private work should compile into public proof
- small tools beat vague roadmaps
- if it does not ship, it is just expensive journaling
